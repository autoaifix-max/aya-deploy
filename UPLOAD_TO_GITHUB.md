# 📤 رفع الملفات إلى GitHub — خطوة بخطوة

> **المهم:** هذه الخطوات على الويب في Chrome مباشرة

---

## ✅ الخطوة 1️⃣: افتح GitHub

اذهب إلى:
```
https://github.com/autoaifix-max/aya-deploy
```

---

## ✅ الخطوة 2️⃣: اضغط "Add file"

```
في الصفحة الرئيسية للـ repo:

     ┌─────────────────────┐
     │ < Code   Issues ... │
     └─────────────────────┘
            ↓
     ┌─────────────────┐
     │ Add file ▼      │  ← اضغط هنا
     └─────────────────┘
```

---

## ✅ الخطوة 3️⃣: اختر "Upload files"

```
ستظهر قائمة:
   • Create new file
   • Upload files    ← اختر هذا
```

---

## ✅ الخطوة 4️⃣: اسحب الملفات

```
الملفات المطلوبة:

من جهازك اسحب:
  📄 index.html
  📄 js/app.js

إلى منطقة الرفع على GitHub
```

**أين تجد الملفات؟**
```
C:\New folder (8)\aya-handoff\
  ├── index.html          ← اسحب هذا
  └── js/
      └── app.js          ← اسحب هذا
```

---

## ✅ الخطوة 5️⃣: اكتب رسالة الـ Commit

في حقل "Commit message":

```
🚀 النسخة الجديدة مع الرسائل العاطفية
```

أو:
```
Update index.html and app.js with new features
```

---

## ✅ الخطوة 6️⃣: اضغط "Commit changes"

```
الزر الأخضر في الأسفل:
     ┌──────────────────┐
     │ Commit changes   │
     └──────────────────┘
```

---

## ⏳ ماذا يحدث بعدها؟

```
1. GitHub سيقبل الملفات (5 ثوان)
2. Vercel سيرى التحديث تلقائياً (15 ثانية)
3. Vercel سينشر النسخة الجديدة (30-60 ثانية)
4. الرابط سيتحدث: https://aya-deploy-blond.vercel.app

✅ خلال دقيقة: التطبيق محدّث!
```

---

## ✅ التحقق من النجاح

بعد انتظار دقيقة:

```
1. افتح:
   https://aya-deploy-blond.vercel.app

2. اضغط F5 (تحديث الصفحة)

3. يجب أن ترى:
   ✓ الرسائل العاطفية من أحمد والبيبي
   ✓ الأزرار الجديدة
   ✓ كل الميزات الجديدة

✅ إذا ظهر كل شيء = نجح!
```
