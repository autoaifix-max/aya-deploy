## 📋 **ما تحتاج أن تفعله الآن (خطوتان فقط):**

### ✅ **الخطوة 1️⃣: رفع الملفات المعدلة إلى GitHub (أونلاين)**

**الطريقة الآمنة والسهلة:**

```
1. اذهب إلى:
   https://github.com/autoaifix-max/aya-deploy

2. اضغط الزر الأخضر: "Add file" > "Upload files"

3. اسحب أو اختر:
   ✓ index.html
   ✓ js/app.js

4. اكتب رسالة Commit:
   "🎉 النسخة النهائية — جاهزة للنشر"

5. اضغط: "Commit changes"

✅ خلال 1-2 دقيقة Vercel سينشر تلقائياً
```

---

### 🔑 **الخطوة 2️⃣: إضافة ANTHROPIC_API_KEY في Vercel**

**هذا هو المفتاح الأخير:**

```
1. اذهب إلى:
   https://vercel.com/dashboard/aya-deploy-blond

2. اضغط: "Settings" (أعلى اليسار)

3. اختر من القائمة اليسار: "Environment Variables"

4. أضف متغير جديد:
   ┌────────────────────────────────────┐
   │ Name:                              │
   │ ANTHROPIC_API_KEY                  │
   ├────────────────────────────────────┤
   │ Value:                             │
   │ bu_mK_BZpBS8imkk06wJLTyFkvtpPVaW...│
   ├────────────────────────────────────┤
   │ Environments:                      │
   │ ✓ Production                       │
   │ ✓ Preview                          │
   └────────────────────────────────────┘

5. اضغط: "Save"

✅ Vercel سيعيد التشغيل تلقائياً
✅ Claude AI سيعمل بعدها مباشرة
```
