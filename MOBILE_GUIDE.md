# 📱 دليل الجوال — استخدام التطبيق من Samsung Galaxy A25

---

## 🌐 **الخطوة 1: افتح التطبيق**

### على جوالك (Samsung Galaxy A25):

```
1. افتح Chrome أو أي متصفح
2. اذهب إلى:
   https://aya-deploy-blond.vercel.app

3. سيحمّل في ثانية أو اثنتين
```

---

## 🎯 **التبويبات الخمسة:**

### 1️⃣ **تبويب "اليوم" 📅**

```
ماذا تجد هنا:
✓ رسالة أحمد اليومية (متنوعة كل يوم)
✓ رسالة من البيبي (دعم وحنان)
✓ أزرار واتساب السريعة:
  • طمّني أحمد
  • أحتاج حضنك
  • جيب لي ماء
  • أبي أكل خفيف
  • اتصل علي
  • محتاجة أطمئن

كيف تستخدمها:
• اضغط "رسالة ثانية" لنص جديد
• اضغط أي زر واتساب لإرسال رسالة لأحمد
• شاهد الملخص اليومي
```

### 2️⃣ **تبويب "حالتي" 💭**

```
ماذا تجد هنا:
✓ اختر حالتك (غثيان، تعب، دوخة، طبيعي)
✓ أضف ملاحظة (اختياري)
✓ اضغط "احصلي على نصيحة"

⏳ ملاحظة:
• Claude AI معطل الآن (ينتظر مفتاح)
• سيقول: "خدمة Claude معطلة حالياً"
• لاحقاً عندما نضيف المفتاح:
  ستحصلين على نصائح ذكية مباشرة! 🤖
```

### 3️⃣ **تبويب "احتياجات الجسم" 🍎**

```
ماذا تجد هنا:
✓ قائمة العناصر الغذائية (حمض الفوليك، بروتين، إلخ)
✓ عداد الماء الكبير:
  • اضغط "+" لإضافة كوب
  • اضغط "-" لإنقاص
✓ نسبة الإكمال (من 8 أكواب)

كيفية استخدام الماء:
• كل كوب تشربيه اضغطي "+" 
• سيُحفظ تلقائياً
• إذا أغلقتي التطبيق سيبقى محفوظ
```

### 4️⃣ **تبويب "الطبيبة" 👨‍⚕️**

```
ماذا تجد هنا:
✓ اسم العيادة/المستشفى
✓ تاريخ الموعد
✓ الوقت
✓ ملاحظات إضافية

كيفية الاستخدام:
• أدخلي بيانات موعدك
• اضغطي "حفظ"
• سيُحفظ محلياً على جوالك
```

### 5️⃣ **تبويب "التنبيهات" 🔔**

```
ماذا تجد هنا:
✓ تحذيرات طبية مهمة
✓ الأشياء المحظورة أثناء الحمل
✓ معلومات صحية موثوقة

ملاحظة طبية:
• هذا التطبيق مساعد فقط
• ليس بديلاً عن الطبيبة
• استشيري الطبيبة دائماً في الحالات الطارئة
```

## 🌙 **ميزة إضافية: Dark Mode**

```
في الزاوية العلوية اليسار:
• اضغطي على 🌙 (Crescent Moon)
• سيتحول التطبيق إلى الوضع الداكن
• اضغطي على ☀️ للعودة للوضع الفاتح
```

## 💬 **زر الواتساب الرئيسي**

```
في الزاوية السفلى اليسار:
• اضغطي "أحتاجك يا أحمد 💬"
• سيفتح واتساب تلقائياً
• يمكنك إرسال رسالة مخصصة أي وقت
```

---

## 📥 **تثبيت التطبيق على الجوال (PWA)**

```
لجعله يعمل بدون انترنت:

1. افتح المتصفح وادخل الرابط
2. اضغط على القائمة (ثلاث نقاط أعلى يمين)
3. اختر "إضافة التطبيق للشاشة الرئيسية"
4. اضغط "إضافة"

النتيجة:
✓ أيقونة التطبيق على الشاشة الرئيسية
✓ يعمل كتطبيق عادي
✓ يعمل بدون انترنت (بدون Claude)
```
